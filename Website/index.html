In week 2 I experienced my first lectures of my course, this was really exciting for me as i haven't done any computer studies before joining this university. I met all of my module tutors, where some classes were more interesting than others, I really enjoyed my web development lab class as I got to interact and start coding with HTML5 and by the end of that session I had a 3 page basic website.

In this class I learned how to:

  1. Set up a folder for my websites on my Web (W:) workspace
  2. Link three web pages together with a navigation menu using a relative link
  3. Create links to external websites using an absolute link
  4. Use h1 and h2 tags
  5. Edit the page title
  6. Validate HTML using the W3C validator
